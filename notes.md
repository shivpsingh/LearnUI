# CSS Notes

```css
h2 + a {
    /* All a tags followed by h2 */
}

textarea ~ button {
    /* all button followed by textarea they are in same parent */
}

ul > li {
    /* Every child */
}

ul li {
    /* Descendents */
}

h2[attr=value] {
    /* Attribute Selector */
    /* = - equal, ^= - startsWith, $= - EndsWith, *= - Contains*/
    /* ~= - contains seperated by whitespace */
    /* |= - select everything with contains value seperated with white space and followed by '-' */
}

body {
    background: url() cover;
}

```
